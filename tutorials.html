Limut Tutorials

About these tutorials

Limut is a live coding system, where you can create music and visuals live by writing code. These tutorials are designed to provide a gentle introduction to the concepts and syntax of limut.

Use them by opening Limut in a separate tab, and then copying the code snippets in. Always experiment with the snippets! Change numbers, combine multiple snippets and generally play around. Its the best way to learn.

Under the main Limut console are various sections of documentation on Limut syntax, synths and params. They are there to be used at any time as a reference and for ideas. The first section of documentation contains examples ready to be copied into the Limut code editor, played, and modified.

Further examples of live coding with Limut can be found on YouTube; for example here. By following along as the code is built up, you can understand new concepts and pick up ideas.

Then, hit the play button that looks like this: . You should hear a bass riff start to play. To stop the music hit the stop button: . You can also use keyboard shortcuts; Ctrl-Enter to play and Ctrl-. to stop.

Unlike some live coding systems, in Limut pressing play runs all the code in the editor, and nothing persists apart from the code that is run. This means you can stop a player by deleting it or commenting it out and then hitting play. Try it now; place a // comment at the beginning of the line, either by typing or hitting the 'toggle comment' button . Then hit play, and the bass will stop.
